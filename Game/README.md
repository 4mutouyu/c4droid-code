# 游戏项目区(Game)

---

### 说明

&emsp;&emsp;本区域存放C4droid的游戏项目，项目不限于原创，但必须标注来源和作者。

### 位置

+ **Console控制台** - 存放在Console文件夹下

+ **Allegro图形库** - 存放在Allegro文件夹下

+ **NativeActivity** - 存放在NDK文件夹下

+ **SDL1.x** - 存放在SDL文件夹下

+ **SDL2.x** - 存放在SDL2文件夹下

+ **SFML** - 存放在SFML文件夹下 

+ **FLTK** - 存放在FLTK文件夹下

+ **Qt** - 存放在Qt文件夹下
